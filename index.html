¿Qué es bootstrap?

Es un framework de CSS, el cual contiene miles de herramientas prediseñadas para el desarrollo de sitios web adaptables, mejorando y agilizando nuestro trabajo.

En esta materia aprenderemos a utilizar lo básico de boostrap para diseñar un sitio web.
